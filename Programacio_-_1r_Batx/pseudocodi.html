Pseudocodi

El pseudocodi ens permet expressar un algoritme amb una estructura semblant a les dels llenguatges de programació sense haver-nos de preocupar excessivament per la sintaxi ja que no hi haurà cap màquina que l'hagi d'interpretar.

Els algoritmes creats amb pseudocodi poden incloure diferents elements:

  • Variables on s'emmagatzemen les dades mentre dura l'execució
  • Operacions
    • De lectura i escriptura de dades
    • Aritmètiques
    • Lògiques
    • D'assignació d'un valor a una variable
  • Condicionals i iteracions

Sintàxi

  1. Sempre iniciarem l'algoritme amb iniciPrograma i el nom del programa en qüestió sense espais
  2. Sempre iniciarem l'algoritme amb fiPrograma
  3. Utilitzarem les tabulacions per indicar la jerarquia del codi dins el programa, els condicionals i les iteracions
  4. Utilitzarem les operacions escriure i llegir per a l'entrada i la sortida de dades del programa
