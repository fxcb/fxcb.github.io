06 - L'evolució de la web. Formularis HTML

Web 2.0

El concepte de web 2.0 s'encunya l'any 2006 per identificar un conjunt de pràctiques i tecnologies relacionades amb la creació de pàgines web que incorporen la voluntat de transformar al visitant en un agent actiu d'Internet ja sigui facilitant la publicació de continguts com incorporant funcionalitats d'interacció amb el contingut (comentaris, m'agrada, compartir...).

Apareixen en aquest moments els portals de blogs, les wikis i les primeres xarxes socials (mySpace, Facebook...)

Activitat

Investigueu a través d'aquesta publicació o fent recerca per Internet quines són les característiques de l'anomenada web 3.0

Formularis HTML

Per tal de facilitar la publicació de continguts a la web, és molt comú, inclús avui en dia utilitzar algun tipus de formulari en que l'usuari introdueix la informació que vol compartir.

El llenguatge HTML incorpora la possibilitat de dissenyar formularis amb diferents tipus de camps (text, selectors, checkbox, radio button...)

A la web de referència W3schools es pot consultar tota la informació sobre com crear un formulari amb HTML
