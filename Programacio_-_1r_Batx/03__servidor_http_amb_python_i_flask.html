03 - Servidor HTTP amb Python i Flask

Servidor HTTP

El servidor HTTP (Hyper Text Transfer Protocol) és l'encarregat d'atendre les peticions que fan els navegadors web (Firefox, Chrome, Safari... ). De fet, hipertext és un terme per referir-nos al que coneixem com a pàgines web.

Tal com podem apreciar en l'esquema, el navegador fa una petició amb una determinada informació (capçaleres) sobre el que li està demanant i el servidor repon amb unes noves capçaleres junt amb el contingut sol·licitat.

Els navegadors fan bàsicament dos tipus de peticions HTTP, les peticions de tipus GET en què simplement sol·liciten informació (les més comunes), i les peticions tipus POST en què el navegador aporta certa informació al servidor. Per exemple, en el moment de fer login a una pàgina web, el navegador realitza una petició POST en què proporciona el servidor el nom d'usuari i la contrasenya.

Codis HTTP

Els missatges HTTP disposen d'un sistema de codis que indiquen ràpidament l'estat en què es troba la comunicació.

La primera xifra del codi fa referència a la categoria general.

El més comú és que les comunicacions es realitzin correctament i que, per tant, la majoria dels missatges estiguin marcats amb el codi 200.

Per altra banda, segur que en diferents ocasions ens hem trobat amb errors del tipus "404 Not Found" quan sol·licitem una direcció que no existeix o "500 Internal Server Error" quan el servidor falla per algun motiu.

Resposta HTTP

El servidor HTTP retorna tota la informació necessària perquè el navegador pugui mostrar correctament la pàgina web sol·licitada.

Les pàgines actuals incorporen informació codificada en tres tipus de llenguatges diferents. Codi HTML que incorpora l'estructura (esquelet) dels continguts, codi CSS que aporta a aquests continguts un aspecte estètic determinat (colors, tipografies...) i codi Javascript que, a través de la programació, fan que els continguts es comportin d'una certa manera (p.ex. animacions).

Abordarem en detall aquests llenguatges més endavant.

Flask

Flask és un entorn de treball per a aplicacions web escrit en Python que permet executar un petit servidor web que aten peticions HTTP i retorna majoritariament fitxers HTML, tot i que també pot retornar dades directament.
